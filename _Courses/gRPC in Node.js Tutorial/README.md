## WHAT IS gRPC?

Developed by Google
Uses HTTP/2 and Protocol Buffers (binary)
Supports streaming (client, server, bidirectional)
Auto-generates client/server code

## WHAT IS REST

Based on HTTP 1.1
Text-based (usually JSON)
Widely adopted, language-agnostic
Stateless, standard verbs: GET, POST, PUT, DELETE

## WHEN USE gRPC?

Real-time data needs
Microservices
Performance-critical systems
Mobile and IoT
